Triangulum

A top down scrolling space action game!

Download .zip Download .tar.gz View on GitHub

A top down scrolling space action game! Asteroids, lasers, bosses, cool music and explosions. A classic space shooter that wants to be the next R-type!

Dependencies

  • SFML 2.1
  • EntityX (latest)
  • yaml-cpp (0.3.0)

Prepare to build

  • Follow EntityX instructions for installation
  • Install yaml-cpp 0.3.0
  • Install SFML 2.1

Building

  • git clone
  • cd Triangulum
  • mkdir build
  • cd build
  • cmake .. && make
